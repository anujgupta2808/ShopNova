# ShopNova Ecommerce Homepage

This folder contains a static ecommerce landing page for **ShopNova**, a premium online store homepage built with HTML, CSS, and JavaScript.

→ What's Included

- Responsive navigation bar with mobile menu support
- Hero section with featured promo content
- Category showcase
- Featured products grid with filtering tabs
- Deal banner with countdown timer
- Customer testimonials section
- Newsletter subscription form
- Cart drawer and toast notifications
- Footer with store and support links

→ Files

- `index.html` - Main homepage
- `css/style.css` - Shared page styling
- `js/main.js` - Interactive behavior such as menu toggle, cart, filtering, countdown, and newsletter handling
- `images/` - Project assets, if added later

→ Dependencies

The page uses external libraries loaded from CDNs:

- Google Fonts: Inter
- Font Awesome icons

→ How to Run

1. Open `index.html` in a browser.
2. Make sure the relative paths to `css/style.css` and `js/main.js` stay intact.
3. If you are using the linked pages such as `shop.html`, `categories.html`, `deals.html`, or `checkout.html`, keep them in the same folder so navigation works correctly.

→ Notes

- This is a static front-end demo, not a full ecommerce backend.
- Some footer links reference pages that may not exist yet, such as `about.html`, `faq.html`, `returns.html`, and `shipping.html`.
- Product data is hardcoded in the HTML and can be edited directly in `index.html`.
